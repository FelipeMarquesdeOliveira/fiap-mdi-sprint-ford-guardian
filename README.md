# Ford Guardian - Sprint Mobile

**Repositório da Sprint de Mobile Development & IoT - Ford x FIAP**

## Projeto: Ford Guardian

**Desafio 02 - Impulsionando o VIN Share na América do Sul com Soluções Inteligentes**

Este repositório contém o código-fonte do aplicativo mobile Ford Guardian, desenvolvido como parte da sprint acadêmica em parceria com a Ford e a FIAP.

### Estrutura

```
fiap-mdi-sprint-ford-guardian/
├── FordGuardian/           # Aplicativo React Native + Expo
│   ├── src/
│   │   ├── presentation/  # Screens, Components, Navigation
│   │   ├── domain/         # Entities
│   │   ├── data/          # Repositories, Mocks
│   │   ├── infrastructure/ # API services, Storage
│   │   └── shared/        # Theme, Constants
│   ├── App.tsx
│   └── README.md          # Documentação completa do app
├── sprint-info/           # Informações da sprint (gitignored)
└── README.md               # Este arquivo
```

### Como Executar

```bash
cd FordGuardian
npm install
npx expo start
```
